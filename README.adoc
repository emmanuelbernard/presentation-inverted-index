== How to build

[source]
----
# Generate the HTML
./gradlew asciidoc

# Listen to file change and regenerate the HTML

./gradlew -t asciidoc

# Generate the PDF
./gradlew generatePdf

# more tasks
./gradlew tasks --all
----
